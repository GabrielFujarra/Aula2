Tags

Tags em HTML são comandos usados para marcar e estruturar o conteúdo de uma página web. Elas indicam ao navegador como exibir textos, imagens, links, listas, tabelas e outros elementos. Cada tag geralmente tem uma tag de abertura P e uma tag de fechamento /P, envolvendo o conteúdo que será formatado.

Listas

Sao separadas em dois tipos, ordenadas OL e nao ordenadas UL. Para cada elemento da lista usamos o LI

Imagens e Links

Para exibir imagens usamos a tag IMG e para exibir links usamos a tag A e a tag HREF
